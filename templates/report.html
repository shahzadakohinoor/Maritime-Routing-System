Project Report

Project Title

AI-Powered Maritime Weather Routing & Voyage Optimization System

Abstract

This project is a web-based maritime route optimization system that helps shipping operators analyze voyage time, fuel consumption, weather risk, voyage cost, laycan risk and route diversion decisions. The system provides a digital dashboard for safer and more economical ship routing.

Objectives

  • Calculate voyage duration using distance and vessel speed.
  • Estimate fuel consumption and voyage cost.
  • Identify weather risk zones along the route.
  • Suggest alternate detour routes during bad weather.
  • Display route and weather information using an interactive map.

System Modules

  • Route Planner
  • Weather Risk Dashboard
  • Fuel Consumption Calculator
  • Voyage Cost Calculator
  • Interactive Route Map
  • Route Diversion Recommendation

Technology Used

Part Technology
Frontend HTML, CSS, JavaScript
Backend Python Flask
Map Leaflet.js
Database MySQL

Future Scope

  • Real-time marine weather API integration.
  • AI-based route recommendation engine.
  • PDF voyage report generation.
  • Fleet management dashboard.
  • Carbon emission calculation.
