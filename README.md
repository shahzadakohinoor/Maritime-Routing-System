# Maritime Routing System

AI-Powered Maritime Weather Routing & Voyage Optimization System is a Flask-based web application for voyage route planning, weather risk analysis, fuel cost calculation, route diversion, vessel registration, and PDF report generation.

## Features

- Voyage route calculation
- Fuel consumption estimation
- Voyage cost analysis
- Weather risk dashboard
- Interactive route map
- Voyage history
- Vessel registration
- Admin login dashboard
- PDF voyage report download
- MySQL database support

## Technology Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Leaflet.js
- MySQL
- XAMPP
- ReportLab
- Open-Meteo Marine Weather API
